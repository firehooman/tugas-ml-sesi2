# Tugas Machine Learning Sesi 2 - Titanic Dataset

## Deskripsi
Repositori ini berisi latihan manipulasi data menggunakan library Pandas dan Scikit-Learn pada dataset Titanic.

## Apa yang saya pelajari:
* **Data Loading**: Membaca dataset dari URL GitHub.
* **Data Cleaning**: Menangani *missing values* dengan median (untuk Age) dan menghapus baris kosong.
* **Preprocessing**: Menggunakan *One-Hot Encoding* untuk data kategorikal dan *MinMaxScaler* untuk normalisasi skala data.
